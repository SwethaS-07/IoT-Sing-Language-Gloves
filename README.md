IoT Enhanced Sign language Glove: Bridging Speech,Touch and Gestures

Overview

Sign Glove is an IoT-based smart glove designed to help people with speech and hearing impairments communicate more effectively. The glove uses flex sensors to recognize hand gestures and converts them into text and voice output. It also connects to an Android application through Bluetooth for real-time communication.

Features

- Real-time hand gesture recognition
- Text display on LCD
- Voice output using DFPlayer Mini
- Bluetooth communication with Android application
- Mobile app for displaying recognized gestures
- Easy-to-use and portable design

Hardware Components

How to Run

1. Upload the Arduino code to the Arduino Nano.
2. Connect all hardware components according to the circuit diagram.
3. Install and open the Android application.
4. Pair the phone with the HC-05 Bluetooth module.
5. Perform hand gestures to view the corresponding text and hear the voice output.
